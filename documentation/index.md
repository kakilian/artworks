# Artworks Documentation Hub

Welcome to the documentation for **Artworks**, a curated e-commerce platform celebrating original art and visual storytelling. This documentation provides insight into the design decisions, development process, testing, and future potential of the project.

---

## 📂 Overview of Sections

- [UX & UI Design](ux_ui_design.md)  
  Wireframes, layout planning, accessibility considerations, and visual design choices.

- [Testing](testing.md)  
  Manual test cases, browser/device compatibility, bugs encountered, and resolved.

- [Deployment](deployment.md)  
  Setup steps for local development and production hosting on Render with Cloudinary integration.

- [Future Improvements](future_improvements.md)  
  Planned features, enhancements, and post-submission development goals.

- [Acknowledgements](acknowledgements.md)  
  Asset credits, inspiration sources, and support received throughout the project.
